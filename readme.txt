Adds integration between WordPress/WooCommerce and Granatum Financeiro.

== Description ==
<strong>WooCommerce Granatum</strong> enables automatic integration between your online store WooCommerce made with WordPress and your account in <strong>Granatum Financeiro</strong>, making it easier financial management of your company.

The Granatum is a financial control online software created especially for micro and small Brazilian companies that offers many features, including: cash flow, accounts payable and receivable, management reports and Goals.

The plugin WooCommerce Granatum was developed independently and the developer of this plugin does not have connections with the company that owns the Financial Granatum.

This plugin was developed from the official documentation of Granatum and uses the latest version of the API integration.

== Installation ==
<ul>
<li>Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;</li>
<li>Activate the plugin;</li>
<li>Navigate to "Main Menu" -> "WooCommerce" -> "Granatum"</li>
<li>On the page that was opened enter your Token API</li>
</ul>
You find your API Token on the sidebar settings page of your company in Granatum.

<h3>Instalação e configuração em Português:</h3>
<ul>
<li>Envie os arquivos do plugin para o diretório de plugins, ou instale usando a opção "Adicionar novo plugin" nativa do WordPress;</li>
<li>Ative o plugin;</li>
<li>Navegue até o "Menu Principal" -> "WooCommerce" -> "Granatum"</li>
<li>Na página que foi aberta entre com sua Token API</li>
</ul>
Você encontra sua Token API na barra lateral da página de configurações da sua empresa no Granatum.
